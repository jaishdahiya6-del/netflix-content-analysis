## 🎯 Project Goals (Day 3 Update)
The core objective of this project is to decode Netflix's content library to understand:
* **Market Dominance:** Identifying which content type (Movies vs. TV Shows) yields higher volume.
* **Global Footprint:** Mapping out which countries are the biggest content hubs.
* **Content Gap Analysis:** Finding underserved genres or ratings to suggest future investments.
* **Personalization:** Building a recommendation engine to mimic the Netflix user experience.

---

## 📌 Project Overview
This project performs a complete analysis of **8,800+ Netflix titles** to uncover strategic insights about audience targeting and global expansion.

* **Dataset:** [Netflix Movies and TV Shows — Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
* **Methodology:** EDA -> Data Cleaning -> Visualization -> NLP (TF-IDF) -> Clustering.

## 🛠️ Data Science Workflow

### 1. Exploratory Data Analysis (EDA)
- **Cleaning:** Handling null values in `director` and `cast` columns.
- **Trend Analysis:** Visualizing the shift from cinema to streaming originals.

### 2. Machine Learning Pipeline
* **Content Recommender:** Using **TF-IDF Vectorization** and **Cosine Similarity** to suggest content based on descriptions.
* **Clustering:** Applying **K-Means** to group titles by common themes.
